💎 VendaSync Evolution - Sistema de Gestão Integrado
Esta é a evolução final do projeto, onde a aplicação deixou de ser apenas visual para se tornar um sistema Full Stack completo, seguindo padrões de mercado.

🚀 Evolução e Melhorias (Atividade 2)
Arquitetura MVC: Divisão organizada em Models, Views e Controllers, permitindo escalabilidade.

Back-end em Node.js: Servidor configurado com Express para gerenciar as regras de negócio e rotas da API.

Persistência de Dados: Implementação de armazenamento em arquivos JSON, garantindo que os dados sejam salvos permanentemente no servidor.

CRUD Completo: Possibilidade de Criar, Ler, Atualizar e Deletar produtos via métodos HTTP (GET, POST, PUT, DELETE).

UI Premium: Evolução do layout para um design Dark Mode/Glassmorphism com Tailwind CSS.

🛠️ Tecnologias e Conceitos (Checklist Técnico)
Node.js & Express: Servidor e API REST.

POO (Programação Orientada a Objetos): Classes para representação de entidades.

Módulo FS: Manipulação de arquivos do sistema para banco de dados JSON.

Async/Await: Tratamento de requisições assíncronas.

📁 Como Rodar
Instale as dependências: npm install

Inicie o servidor: node server.js

Acesse: http://localhost:3000
